# Gia Phả Dòng Họ (Minimal React)

Simple local React app (no backend). Data lives in `/data/data.json`.

How to run (Windows PowerShell):

```powershell
npm install
npm run dev
```

Files added:
- `data/data.json` — sample family data (≥3 đời)
- `src/components/*` — Header, GenealogyTree, MemberDetail, Search, DeathAnniversary
- `src/App.jsx`, `src/index.jsx`, `src/styles.css` — app wiring and styles
